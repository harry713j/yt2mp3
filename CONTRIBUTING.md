# 🤝 Contributing to YT2Mp3

First off, thanks for taking the time to contribute! 🎉

---

## 🐛 Reporting Bugs
- Check the **issues tab** first to avoid duplicates.
- Provide as much detail as possible (error logs, screenshots, steps to reproduce).

---

## ✨ Feature Requests
- Open an issue with the label `enhancement`.
- Describe why the feature would be useful and how it should work.

---

## 🔧 Development Setup

1. Fork the repo
2. Clone your fork:
```bash
   git clone https://github.com/harry713j/yt2mp3.git
```
## Install dependencies:

```bash
cd server
npm install
# and
cd client
npm install
```

## Run the development server:
```bash
cd server
npm run dev
# and
cd client
npm run dev
```
## 📦 Submitting Changes

Create a new branch:
```bash
git checkout -b feature/my-awesome-feature
```

Commit your changes:
```bash
git commit -m "feat: add my awesome feature"
```

Push the branch:
```bash
git push origin feature/my-awesome-feature
```

Open a Pull Request 🎉

📝 Commit Guidelines

Use Conventional Commits
 for messages:
```text
feat: → new feature
fix: → bug fix
docs: → documentation changes
refactor: → code restructuring
chore: → maintenance
```

Example:
```text
feat: add duration validation for videos
```
